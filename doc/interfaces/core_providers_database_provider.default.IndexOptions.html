expires utilizes the ms module from guille allowing us to use a friendlier syntax:

example
const schema = new Schema({ prop1: Date });

// expire in 24 hours
schema.index({ prop1: 1 }, { expires: 60*60*24 })

// expire in 24 hours
schema.index({ prop1: 1 }, { expires: '24h' })

// expire in 1.5 hours
